# Unit Converter

A simple and responsive Unit Converter built with **HTML, CSS, and JavaScript**. This application allows users to quickly convert between Metric and Imperial units for length, volume, and mass.

## Features

* Convert **Meters ↔ Feet**
* Convert **Liters ↔ Gallons**
* Convert **Kilograms ↔ Pounds**
* Instant conversion results
* Clean and responsive user interface
* Built using vanilla JavaScript with no external libraries

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

1. Enter a number in the input field.
2. Click the **Convert** button.
3. The application calculates and displays:

   * Length conversions (Meters and Feet)
   * Volume conversions (Liters and Gallons)
   * Mass conversions (Kilograms and Pounds)

## Conversion Rates

| Unit Type           | Conversion Rate           |
| ------------------- | ------------------------- |
| Meters to Feet      | 1 meter = 3.281 feet      |
| Feet to Meters      | 1 foot = 0.305 meters     |
| Liters to Gallons   | 1 liter = 0.264 gallons   |
| Gallons to Liters   | 1 gallon = 3.785 liters   |
| Kilograms to Pounds | 1 kilogram = 2.204 pounds |
| Pounds to Kilograms | 1 pound = 0.454 kilograms |
